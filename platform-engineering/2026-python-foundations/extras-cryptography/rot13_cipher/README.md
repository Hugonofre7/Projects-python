# ROT13 Cipher

Cifrado de sustitución simple que rota las letras 13 posiciones en el alfabeto. Proyecto #61 del libro "The Big Book of Small Python Projects".

## 🔧 Cómo funciona

ROT13 es un cifrado simétrico donde:
- A → N, B → O, ..., M → Z
- N → A, O → B, ..., Z → M

Aplicar ROT13 dos veces devuelve el texto original.
